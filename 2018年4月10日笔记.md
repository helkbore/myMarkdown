## 3. java 格式化数字(前面自动补零)
```java
int i = 5;
// i = 15;
String str = String.format("%02d", i);  // 共两位不足补0
System.out.println(str);

String str = String.format("%04d", i);  // 共四位不足补0
System.out.println(str);
```
